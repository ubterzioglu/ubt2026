# UI Improvements Plan

## Overview
Portfolio sitesinde 5 ana bölümün görsel ve yapısal iyileştirilmesi.

---

## 1. Key Achievements - Tüm maddeler "büyük sayı + açıklama" formatında

### Mevcut Durum
- 4 adet `achievementHighlights` (value/label/detail formatında - doğru)
- 5 adet `achievementBullets` (düz metin - yanlış format)

### Yapılacaklar
- [ ] `content/profile.ts`: `achievementBullets` dizisini `AchievementItem` formatına çevir
- [ ] `app/page.tsx`: Tüm achievement'ları tek bir grid içinde göster (md:grid-cols-2)
- [ ] Her kart: büyük sayı + altında label + altında detail açıklaması

### Veri Dönüşümü (Anlamlı sayılar)
```
"Implemented Ranorex automation for three major Daimler systems..." →
{ value: "3", label: "major systems automated", detail: "Implemented Ranorex automation for three major Daimler systems with Jenkins-based CI/CD support." }

"Reduced execution time by 40% by optimizing 1,000+ automated checks." →
{ value: "40%", label: "execution time reduced", detail: "Optimized 1,000+ automated checks for faster feedback loops." }

"Led HP ALM to Jira / Xray migration initiatives..." →
{ value: "2", label: "major migrations led", detail: "HP ALM to Jira/Xray migration initiatives that lowered manual effort and operational cost." }

"Onboarded, mentored, and led 30+ QA colleagues..." →
{ value: "30+", label: "QA colleagues mentored", detail: "Across Daimler and Swisslog engagements with hands-on guidance and knowledge transfer." }

"Created KPI-based reporting that improved visibility..." →
{ value: "100s", label: "hours saved monthly", detail: "KPI-based reporting that improved visibility and saved hundreds of hours monthly." }
```

## 4. Experience - Timeline akışı

### Mevcut Durum
- Basit stacked cards

### Yapılacaklar
- [ ] `app/page.tsx`: Timeline tasarımı uygula
  - Sol tarafta dikey çizgi (border-left)
  - Her experience kartının solunda nokta/daire (timeline marker)
  - Kartlar çizgi boyunca aşağı doğru akar
  - Responsive: mobilde çizgi solda, kartlar sağda

### Timeline Tasarım Öğeleri
- Dikey çizgi: `border-l-2 border-accent/30`
- Timeline noktası: `w-3 h-3 rounded-full bg-accent`
- Kartlar: çizginin sağına yerleşik
- **Şirket logoları:** Her kartın sol üstünde şirket logosu
